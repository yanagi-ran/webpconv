## 機能
pngとかjpegとかの画像ファイルをwebp形式に変換してくれます。  
ffmpegではなく公式の `cwebp` を使っています。  
保存先も指定できます。指定しない場合はexeファイルがあるところに。  
処理の速度が速すぎてエラー起きそうなので、ファイルの処理は1秒ごとにしています。
